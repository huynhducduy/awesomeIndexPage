    • NEWS

    Open .../news/news.xml


    date : the date of the event, 2 lines possible with a <br> tag.html supported*.

    title : title of your news, html supported*.

    short : short text under the title of your news, html supported*.

    long : long text of your news, html supported*.If the text is too long, a scrollball will appear.

    NOTE : if you leave blank, <long>, the news will not be clickable to read the rest.


    *Put some html in your texts :


    You can use html text with CDATA.
    How does it work ?
    <title><![CDATA[ your html text...]]></title>
    Instead of simply type your description, you have to add these tag when opening : <![CDATA[ and these tag when closing ]]>
    Between those two tags you can use html balises.
    Examples of HTML Tags :
    change the color of the text :
    <font color="#FF0000">your text</font>
    #FF0000 is the hexadecimal value for Red
    make a link :
    <a href="http://www.flashden.net" target="_blank">Your link text</a>
    jump a line : <br>
    underline text : <u>you underline text</u>
