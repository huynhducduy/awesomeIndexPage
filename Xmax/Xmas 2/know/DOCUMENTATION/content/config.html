    • The first thing to do is to fill in the config.xml file.

    config.xml looks like that :



    1. The first line :


    bgColor : primary color in hexadecimal (ex : 0x000000), this will the color of the background, and of the pannel.

    handlerOpacity : between 0 and 100, the opacity of the main handler background.

    textColor : secondary color in hexadecimal (ex : 0x000000), this will be the color of the texts, of the elements (arrows, lines,...).

    logoUrl : this is the location of picture used as logo.The supported formats are : jpg,png, gif and swf.

    defaultBg : this is the location of the default background.The supported formats are : jpg,png, gif and swf with an opaque background.

    Note : all background images have to be the same dimensions

    bgMusic : set on true if you want to play a music in the site.The file must be a mp3 file, placed on the same level than the
    main.swf, and it must be named "music.mp3".The other value is false.

    startSound : set on true if you want the music automaticaly plays when entering the site.The other value is false.


    2.Name of your site:

    <nameOfYourWebsite></nameOfYourWebsite> : name of your website for the deeplinking.That will appear as the name of your browser window.


    3.Adding a module :

    <rub>

    <title>home</title>

    <folder>modules/home/</folder>

    <background>modules/home/home_bg.jpg</background>
    </rub>

    title : the name of your topic.

    folder : the location of the module used, IMPORTANT : be careful to end this by "/", the name of the swf comes after, and it will produce an error if you forget the "/".

    backround : location of the image used as background for this topic,if you leave it empty, the default background will be used. The supported formats are : jpg,png, gif and swf with an opaque background.

    Note : all background images have to be the same dimensions


    You can add as many topics you want, if there is too much topic, they will be automatically scrolled.


    4.External link case :


    In case you need to make a direct link to a location outside of the site, it's possible.


    <rub>
    <title www="http://www.your-link.com">external link</title>

    <folder></folder>

    <background></background>
    </rub>
    Just add in the title node an attribute called www="http://www.your-link.com".

    You don't need to fill folder and background.


    5. Bottom menu (twitter, rss) :


    To add the twitter icon and the rss feed icon :



    Just fill with your twitter account name, and the URL of your RSS feed.

    NOTE : if you leave these field blank, the icons will not be displayed.
